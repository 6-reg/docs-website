New Relic 사용자가 New Relic 조직에 로그인할 수 있는 방법에는 여러 가지가 있습니다.

* 이메일 및 비밀번호: 기본 로그인 방법입니다. 여기에는 이메일 주소와 비밀번호 입력이 포함됩니다.
* Single Sign-On: ID 공급자 로그인 옵션은 [SAML SSO 및 SCIM 소개를](/docs/accounts/accounts-billing/new-relic-one-user-management/introduction-saml-scim) 참조하십시오.

로그인에 문제가 있습니까? [로그인 문제 해결](/docs/accounts/accounts-billing/account-setup/troubleshoot-new-relics-password-email-address-login-problems/) 을 참조하십시오.

사용자 로그인 방법을 관리하려면 [인증 도메인](/docs/accounts/accounts-billing/new-relic-one-user-management/authentication-domains-saml-sso-scim-more) 을 참조하십시오.

## 조직에서 사용자를 추적하는 방법 [#email]

[New Relic 조직](/docs/accounts/accounts-billing/account-structure/new-relic-account-structure) 은 여러 계정을 가질 수 있으며[사용자 관리 솔루션](/docs/accounts/accounts-billing/new-relic-one-user-management/introduction-managing-users) 을 사용하여 사용자를 하나 이상의 계정에 할당할 수 있습니다. 이메일 주소로 사용자를 추적하므로 사용자를 여러 계정에 추가하는 경우 해당 사용자 기록에 동일한 이메일 주소가 있는 한 단일 사용자로 취급됩니다. 이에 대한 자세한 내용은 [사용자 청구](/docs/accounts/accounts-billing/new-relic-one-pricing-billing/user-count-billing) 를 참조하십시오.

## 사용자 레코드 간 전환 [#switch]

둘 이상의 사용자 레코드에 대한 액세스 권한이 있는 사용자의 경우 [사용자 레코드 간 전환을](/docs/accounts/accounts-billing/account-setup/multiple-logins-found) 참조하십시오.
