シングル ページ アプリ (SPA) の監視には、 [ブラウザー エージェントの既定のインストールが](/docs/browser/browser-monitoring/installation/install-browser-monitoring-agent/)付属しています。

## 要件 [#requirements]

[コンパティビリティと SPA モニタリングの要件については、こちら](/docs/browser/single-page-app-monitoring/get-started/spa-compatibility-requirements/) をご覧ください。

New Relic アカウントで最初のモニターアプリを設定する際には、利用規約に同意する必要があります。利用規約に同意することで、New Relic が URL からハッシュフラグメントを収集することを許可します。チェックボックスのオプションを選択するのは、1つのアカウントに対して1回だけです。

## SPAモニタリングの有効化・無効化 [#enable-spa]

[ブラウザー監視を有効にする](/docs/browser/browser-monitoring/installation/install-browser-monitoring-agent#enable)と、デフォルトで SPA 監視が含まれます。これにより、分散トレースを含むさまざまな最新機能にアクセスできるようになります。 一部の古いエージェントのインストールは、アップグレードが必要になる場合があります。[ ブラウザ エージェントの種類](/docs/browser/browser-monitoring/installation/install-browser-monitoring-agent#agent-types)について詳しくは、こちらをご覧ください。
