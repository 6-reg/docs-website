iOSエージェントをインストール、設定、アップグレードするには、アカウント管理者である必要があります。最新バージョンについては、 [リリースノート](/docs/releases/ios) をご参照ください。

## iOSフレームワークの置き換え [#upgrading]

まず、以前のバージョンの iOS エージェント フレームワークを置き換える必要があります。

1. Xcode の Project Navigator (**CMD 1**) から、 **NewRelic.xcframework**を検索します。
2. **NewRelic.xcframework を**右クリックまたはコントロールクリックし、 をクリックして **\[Finder に表示]**を選択します。
3. **NewRelic.xcframework を** ゴミ箱にドラッグします。
4. Xcode プロジェクトが **NewRelic.xcframework** への参照を赤で強調表示していることを確認します。
5. **NewRelic.xcframework を**右クリックまたはコントロールクリックし、 をクリックして **\[削除]** を選択し、古い参照をプロジェクトから削除します。

次に、 **[one.newrelic.com](https://one.newrelic.com/all-capabilities) > Mobile > (アプリを選択) > Settings > Installation**の UI でインストール プロセスを続行します。
