あなたは Geek's Movie Shop の新しい IT エンジニアです。このサイトは、分散インフラストラクチャ全体に展開されています。あなたの前任者はサイトのサーバーを計測していないので、サーバーがどのように動作しているかについてはよくわかりませんが、最近のユーザーの苦情は、システムの観察を開始する時期であることを示しています。

このラボでは、New Relic インフラストラクチャ モニタリングを使用してサーバーを観察し、問題を特定して軽減できるようにします。

## 目的

* ホストに New Relic インフラストラクチャ エージェントを装備する
* インフラストラクチャと関連サービスから洞察を収集する
* インフラストラクチャの根本原因の問題を特定する

## 要件

* 無料の [New Relic アカウント](https://newrelic.com/signup)

  このトラックは、エフェメラル仮想マシンを利用しています。その結果、トラック全体を一度に完了する必要があります。そうしないと、時間切れになると進行状況が失われます。
</Callout>
