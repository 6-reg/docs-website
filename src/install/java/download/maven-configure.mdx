headingText: Configure your `pom.xml` to download `newrelic-java.zip`
---

You need to establish a line of communication between Maven and New Relic. To do that, go into your Maven `pom.xml` file so Maven knows to download the `newrelic-java.zip`. For example, your `pom.xml` file might look something like this: 

```xml
<dependency>
    <groupId>com.newrelic.agent.java</groupId>
    <artifactId>newrelic-java</artifactId>
    <version>JAVA_AGENT_VERSION</version>
    <scope>provided</scope>
    <type>zip</type>
</dependency>
```

Where the snippet reads <var>JAVA_AGENT_VERSION</var>, replace <var>JAVA_AGENT_VERSION</var> with the [latest Java agent version](/docs/agents/java-agent/getting-started/java-release-notes).
