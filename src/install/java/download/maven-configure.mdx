headingText: Configure Maven to download `newrelic-java.zip`
---

To establish a line of communication between Maven and New Relic, you'll need to add a snippet to your `pom.xml`. This snippet cues Maven to download the Java agent with `newrelic-java.zip`. 

Your `pom.xml` file might look something like this: 

```xml
<dependency>
    <groupId>com.newrelic.agent.java</groupId>
    <artifactId>newrelic-java</artifactId>
    <version>JAVA_AGENT_VERSION</version>
    <scope>provided</scope>
    <type>zip</type>
</dependency>
```

Where the snippet reads <var>JAVA_AGENT_VERSION</var>, replace <var>JAVA_AGENT_VERSION</var> with the [latest Java agent version](/docs/agents/java-agent/getting-started/java-release-notes).
