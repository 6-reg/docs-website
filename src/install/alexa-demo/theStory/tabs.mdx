The branching doc requires a bit of thinking power because the ordering isn't exactly intuitive. Here are the main parts of the branching structure:

* The YAML file, which tells the mdx files what order to present themselves
* The front matter of the mdx files
* The text of the mdx file

When you select an image from the above step, then step 4 will change what music/video is presented.
