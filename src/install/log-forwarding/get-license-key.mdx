headingText: Get a license key
---

If you already have a New Relic account, get your <InlinePopover type="licenseKey" /> by going to https://one.newrelic.com/api-keys, click **...** and then **Copy key ID**. 

If you don't already have a New Relic account, [sign up](https://newrelic.com/signup?via=login) and generate a new key by going to the New Relic [API keys page](https://one.newrelic.com/api-keys), click **Create a key**, and then create an ingest-license key type.
